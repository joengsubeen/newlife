초과근무 매식비 정산 변환기

초과근무세부내역 파일을 올리면 성명별 매식비 정산 파일을 다운로드합니다.

엑셀 파일(.xlsx/.xls)을 끌어놓거나 아래에서 선택하세요.
파일을 선택해 주세요.

적용 기준

  • 공휴일 판정: 주말 또는 공휴일 API 기준 (API 실패 시 주말 기준)
  • 제외시간: "HH:MM ~ HH:MM" 형식에서 뒤 시간 - 앞 시간
  • 근무일: (실제퇴근-실제출근-제외시간) 10시간 이상일 때 지급
  • 공휴일: (실제퇴근-실제출근-제외시간) 2시간 이상일 때 지급
  • 출장이 "근무지외출장"으로 시작하면 지급 제외
  • 성명별로 지급가능 일자를 1개 셀에 묶어 출력, 단가 9,000원
